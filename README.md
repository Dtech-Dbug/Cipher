# AskAscii

ASCII Conversions.

- Characters to Integers (`convertToAscii`)
- Integers to Characters (`convertIntegerToChar`)

## Installation using npm and setting up

`npm i AskAscii`

**Setting Up**

```js
const { convertToAscii, convertIntegerToChar } = require("AskAscii");
```
